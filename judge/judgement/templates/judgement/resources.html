  • Download the offline simulator (this is a modified version of the one on the NandToTetris website). Or, if you prefer, you can also compile it yourself from the source code. Run the GUI with: 
    java -jar CPUEmulator-2.5-SNAPSHOT.jar
  • The CPU emulator also has a command line mode that allows you to run test scripts the same way this website does. You just need to run: 
    java -jar CPUEmulator-2.5-SNAPSHOT.jar test.tst 0
    Here, test.tst is your test file (which can be downloaded from this site), and the 0 argument means that your code should run indefinitely without timing out. For production use on this site, some problems have cycle timeouts, and we also time out your code after the simulator has run for a certain number of seconds.
